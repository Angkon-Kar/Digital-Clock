# 🪐 Orbital Clock
# Digital-Clock
A sleek, modern, and interactive clock built with vanilla web technologies. It features three orbital calendar rings (Day, Month, Date), a smooth analog center clock, and a futuristic digital time display with scramble animations.

## ✨ Features

* **Orbital Calendar Rings:** Dynamic, animated rings for the day, month, and date that rotate smoothly into place.
* **Dual Display:** Features both an analog clock face and a digital time readout.
* **Futuristic Text Scramble:** The digital time display scrambles randomly before locking into the correct time upon loading.
* **Interactive Parallax:** The orbital rings react to your mouse movements for a 3D depth effect.
* **Dark & Light Modes:** Seamless theme switching with a beautiful expanding ripple animation.
* **Fully Responsive:** Perfectly scales across desktop, tablet, and mobile devices.

## 🛠️ Installation & Setup

You don't need any complex build tools to run this project. It runs completely in your browser!

### Option 1: Quick Run
1. Download or clone this repository.
2. Open the `index.html` file directly in your web browser.

### Option 2: Running with VS Code (Recommended)
1. Open the project folder in **Visual Studio Code**.
2. Install the **Live Server** extension if you haven't already.
3. Right-click on `index.html` and select **"Open with Live Server"**.
